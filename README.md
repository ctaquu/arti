# arti

- dev database (sqlite) is versioned, but you can run migrations to create your own environment with:
```
yarn bootstrap
```

- local server boot:
```
yarn start-dev
```

- access then @:
```
http://localhost:3000/api/v1/{{whatever}}
```
 
- run tests
```
yarn test
```
